# 🧊 Quito Fresh - ECOS Branding Premium Digital Experience

Este repositorio contiene el ecosistema digital de alta gama para **Quito Fresh**. 
Desarrollado con un enfoque en conversión, lujo visual y automatización de ingresos.

## 🛠 Tech Stack
- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS + Framer Motion (Animaciones 60fps)
- **E-Commerce:** Stripe Billing (Suscripciones Recurrentes)
- **CRM/Email:** Resend + React Email (Recuperación de Carritos)
- **State Management:** Zustand (Ultra-fast UI)

## 📈 Valor Estratégico
1. **Recuperación de Carritos:** Automatización de marketing integrada.
2. **Fresh Pass:** Modelo de suscripción para ingresos predecibles.
3. **Admin Dashboard:** Control total de inventario y analíticas de venta.
